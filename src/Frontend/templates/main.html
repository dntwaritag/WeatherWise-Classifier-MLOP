Welcome to WeatherWise

WeatherWise is an AI-powered weather classification system designed to predict weather conditions based on historical meteorological data from Seattle.

Our model analyzes key weather indicators to provide accurate weather predictions, enabling better planning and decision-making.

Key Features

  • Real-Time Predictions: Instantly analyze weather data and get AI-powered results.
  • Data Insights & Visualizations: Gain insights from key weather features.
  • Bulk Data Upload: Easily upload multiple files for batch predictions.
  • On-Demand Model Retraining: Improve model accuracy with new data.
  • Performance Metrics: Track model accuracy, precision, recall, and F1-score.
